# SpringBoot 多模块项目

## 说明
本项目为SpringBoot多模块整合DEMO,及开发中常用工具类整合

## 架构
* 构建工具：Maven
* 接口文档工具：swagger2
* 内置容器： undertow
* 代码生成器： MyBatis-Plus-Generator
* 定时任务：quartz


## 版本说明
* SpringBoot: 2.1.6.RELEASE
* jdk: 1.8

## 技术选型
* mybatis-plus
* redis
* security
* jwt
* swagger2
* lombok
* mqtt
* webMagic
* quartz
* wxjava


## 发版工具
* jenkins
